# BMI-Project

A simple BMI (Body Mass Index) Calculator built using HTML, CSS, and JavaScript.  
This project calculates BMI based on user input (height and weight) and shows the health category.

## 🚀 Features
- Enter height and weight
- Calculate Body Mass Index
- Shows BMI category (Underweight, Normal, Overweight)
- Simple and responsive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📊 BMI Formula
BMI is calculated using the formula:

BMI = weight (kg) / height² (m)
